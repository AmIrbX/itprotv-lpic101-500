
Commands: 

```
lspci -> lists all recognized PCI devices 
lsusb -> lists all recognized USB devices
lsmod -> lists all installed Modules
rmmod -> Removes a module (Temporary)
```

Directories:

```
/etc/udev #module's naming service
/etc/modprobe.d #Module's probing at the boot
/etc/default/grub #grub configuration file
/etc/grub.d/ #Custom grub configuration
/boot/grub.cfg #grub file that kernel uses (can not change)

```
