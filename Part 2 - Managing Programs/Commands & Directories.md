## Commands: 

```
	nice # Changes Priority Numbers
	top # Monitors and Contorlls Processes
	renice # Changes running programs Execution Priorities
	kill # terminates a process with PID
	kilall # terminates all processes with similar names

	which CMD # Shows the binary path CMD is using
	history # Shows all commands executed after booting
	uname # Information about Operating System
	ps # Shows running processes
	less # Used to read long text files 
	more # like LESS command but only allows scrolling down
	gedit # A GUI for editing text files

	pgrep # Searches by process name and outputs the PID
	bg # Shows all BackGround processes
	fg # Resumes the background program
```
